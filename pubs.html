Research-related open-source contributions
  • Kind equalities in GHC, based on the ICFP'13 paper
  • Visible type application in GHC, based on the POPL'16 submission
  • Glambda, a lambda-calculus interpreter built with GADTs, useful as a case study
  • A new Coercible solver, already part of GHC
  • The units package, a domain-specific type system for dimensional analysis
  • Roles, as based on the ICFP'14 paper, already part of GHC
  • Closed type families, implemented in two major commits, as based on the POPL'14 paper
  • The singletons library, which allows for automatic derivation of declarations that support dependent types in released versions of GHC
