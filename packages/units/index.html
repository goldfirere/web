The units package

This page is the home of the units package, maintained by Richard Eisenberg. This package defines a domain-specific type system for dimensional analysis in Haskell.

Announcements

  • December 18, 2013: units-1.1 released. This version generalized the numerical representation, along with various other small updates to bring it in line with GHC HEAD.
  • August 20, 2013: units-1.0.1 released. This version has the correct base dependency.
